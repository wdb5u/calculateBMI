The program should be able to calculate BMI based off of user input.
Users must be able to input their weight (pounds/kilograms) and height (inches/meters).
Once calculated, the program will print the BMI to the user and also the corresponding weight classification according to https://www.canada.ca/en/health-canada/services/food-nutrition/healthy-eating/healthy-weights/canadian-guidelines-body-weight-classification-adults/quick-reference-tool-professionals.html
